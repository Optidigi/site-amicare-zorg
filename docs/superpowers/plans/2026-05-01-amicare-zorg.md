# Amicare-Zorg One-Pager Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Build the static one-pager described in `docs/superpowers/specs/2026-05-01-amicare-zorg-design.md` and deploy it via Docker/Traefik to `amicare-zorg.nl`.

**Architecture:** Astro 5 minimal starter, no JS framework integrations, Tailwind CSS for styling, self-hosted Fraunces + Inter via `@fontsource-variable`. Three content sections (Hero, Over mij, Contact) plus footer, single supporting Unsplash photo, all on one route. Multi-stage Dockerfile (Node 22 build → nginx:alpine serve). `compose.yml` with Traefik v3 labels for `amicare-zorg.nl` + www→apex redirect, deployed by Coolify.

**Tech Stack:** Astro 5, Tailwind CSS 3 (`@astrojs/tailwind`), `@astrojs/sitemap`, `@fontsource-variable/fraunces`, `@fontsource-variable/inter`, `sharp` (OG image generation), pnpm, Node 22, nginx:alpine.

**Project root for all relative paths:** `/home/shimmy/Desktop/env/amicare-zorg`.
Repo is already initialised at this path (commit history begins with the spec).
All `cd` commands target this directory.

**Verification model:** This site has no JavaScript behaviour to unit-test.
Verification is per-task: `pnpm dev` renders the section as expected,
`pnpm build` succeeds, `pnpm preview` serves the built dist, and a final
`docker run` smoke-test confirms the production image. Lighthouse and visual
inspection in Firefox/Chrome at the end.

|---|---|

- [ ] **Step 2: Write `nginx.conf`**

```nginx
server {
  listen 80;
  server_name _;
  root /usr/share/nginx/html;
  index index.html;

  # Compression
  gzip on;
  gzip_vary on;
  gzip_min_length 1024;
  gzip_types text/plain text/css text/xml application/javascript application/json application/xml image/svg+xml;

  # Hashed Astro assets — long cache, immutable
  location /_astro/ {
    expires 1y;
    add_header Cache-Control "public, immutable";
    try_files $uri =404;
  }

  # Other static assets — modest cache
  location ~* \.(?:jpg|jpeg|png|gif|webp|avif|svg|ico|woff2?)$ {
    expires 7d;
    add_header Cache-Control "public";
    try_files $uri =404;
  }

  # HTML — short cache so deploys propagate fast
  location / {
    expires 5m;
    add_header Cache-Control "public, must-revalidate";
    try_files $uri $uri/ $uri.html /index.html;
  }

  # Astro 404 fallback
  error_page 404 /404.html;

  # Security headers
  add_header X-Content-Type-Options "nosniff" always;
  add_header X-Frame-Options "SAMEORIGIN" always;
  add_header Referrer-Policy "strict-origin-when-cross-origin" always;
}
```

## Task 15: compose.yml with Traefik labels

**Files:**
- Create: `compose.yml`

- [ ] **Step 1: Write `compose.yml`**

```yaml
services:
  web:
    build: .
    image: site-amicare-zorg:latest
    restart: unless-stopped
    networks:
      - coolify
    labels:
      - traefik.enable=true

      # Apex: amicare-zorg.nl on websecure
      - traefik.http.routers.amicare-apex.rule=Host(`amicare-zorg.nl`)
      - traefik.http.routers.amicare-apex.entrypoints=websecure
      - traefik.http.routers.amicare-apex.tls=true
      - traefik.http.routers.amicare-apex.tls.certresolver=letsencrypt
      - traefik.http.services.amicare-apex.loadbalancer.server.port=80

      # www → apex (HTTPS) redirect
      - traefik.http.routers.amicare-www.rule=Host(`www.amicare-zorg.nl`)
      - traefik.http.routers.amicare-www.entrypoints=websecure
      - traefik.http.routers.amicare-www.tls=true
      - traefik.http.routers.amicare-www.tls.certresolver=letsencrypt
      - traefik.http.routers.amicare-www.middlewares=amicare-www-redirect
      - traefik.http.middlewares.amicare-www-redirect.redirectregex.regex=^https?://www\.amicare-zorg\.nl/(.*)
      - traefik.http.middlewares.amicare-www-redirect.redirectregex.replacement=https://amicare-zorg.nl/$$1
      - traefik.http.middlewares.amicare-www-redirect.redirectregex.permanent=true

      # HTTP → HTTPS for both
      - traefik.http.routers.amicare-http.rule=Host(`amicare-zorg.nl`) || Host(`www.amicare-zorg.nl`)
      - traefik.http.routers.amicare-http.entrypoints=web
      - traefik.http.routers.amicare-http.middlewares=amicare-https-redirect
      - traefik.http.middlewares.amicare-https-redirect.redirectscheme.scheme=https
      - traefik.http.middlewares.amicare-https-redirect.redirectscheme.permanent=true

networks:
  coolify:
    external: true
```

Notes:
- The `coolify` external network is what Coolify provisions for Traefik; if your Coolify setup uses a different network name, replace it.
- `letsencrypt` is the conventional resolver name; Coolify exposes it by default. If your Traefik config uses a different name, replace it.
- The `$$1` is intentional — Compose substitutes one `$`, leaving Traefik with `$1`.
